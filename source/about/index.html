Gerry AI is a collection of projects created as a reusable framework of libraries for building and using automated planning algorithms.

The objective of this project is to lower the barrier to entry for people wanting to write or use automated planning algorithms; see our first blog post.

Gerry AI aims to achieve this in the following ways:

  • Fully open source with clear licensing
  • Code hosted on GitHub for ease of access and collaboration
  • Properly packaged for use in Maven projects
  • Coded to be clear to understand, reusable, extensible and fully tested
